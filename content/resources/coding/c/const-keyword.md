# Const Keyword

> [!abstract]
> Const (constant) is a type qualifier that indicates that the data is read-only.

## Rules

- The const variable cannot be left un-initialized at the time of the declaration.
- It cannot be assigned a value anywhere in the program.
- An explicit value needs to be provided to the constant variable at the time of declaration of the constant variable.
