# Settling Time

> [!abstract]
> The settling time is the time elapsed from the application of an step input (ideally instantaneous) to the time at which the system output has entered and remained within a specified error band.

![[resources/control-systems/assets/settling-time.png|500]]

The settling time includes [[Propagation Delay]], plus the time required for the output to stay in close vicinity of the final value.
