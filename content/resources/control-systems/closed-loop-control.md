# Closed Loop Control

> [!abstract]
> Control loop which incorporates feedback and is therefore dependent on the process variable

![[resources/control-systems/assets/closed-loop-control.png]]

## Advantages to Open Loop Control

- disturbance rejection
- guaranteed performance even with model uncertainties
- unstable processes can be stabilized
- reduced sensitivity to parameter variations
- improved reference tracking performance
- improved rectification of random fluctuations
