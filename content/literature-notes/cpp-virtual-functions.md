# VIRTUAL FUNCTIONS
---

>[!abstract]
> A virtual function is a member function in a base class, that can be **overridden** in derived classes.

## Without *virtual*

The base function runs, even if the object is from a child class.


## With *virtual*

The child's version of the function runs, when the object is from a child class.
