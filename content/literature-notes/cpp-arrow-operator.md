# THE -> OPERATOR
---

>[!abstract]
>The `->` operator is used as a shortcut for writing `(*pointer).member`

The `->` operator is used to access members (like functions or variables) through a [[pointer]].

```cpp
foo* a = new Foo();
a->foo_func(); // Same as (*a).foo_func()
```
