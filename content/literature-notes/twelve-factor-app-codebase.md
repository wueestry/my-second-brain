# TWELVE FACTOR APP CODEBASE

---

![[meta/assets/twelve-factor-app-codebase.png]]

A [[twelve-factor app]] is always tracked in a version control system.
There is always exactly one app per repo (codebase).

There are usually multiple running instances (deploys) of the same app (production, staging, development), but all of them share the same codebase.
