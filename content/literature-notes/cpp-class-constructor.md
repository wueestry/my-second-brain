# CLASS CONSTRUCTOR
---

>[!abstract]
>A constructor is a special method that is automatically called when an object of the class is created.

```cpp
class Foo {
	public:
		// Attributes
		Foo(string x) {
			// Parameters used in Initialisation.
			foo = x;
		}
}
```
