# THE -> OPERATOR IN C++

---

The `->` operator is used to access members (like functions or variables) through a [[distilled-notes/cpp-pointer]] and a shortcut for writing `(*pointer).member`

```cpp
foo* a = new Foo();
a->foo_func(); // Same as (*a).foo_func()
```
