# SCOPE

---

The **scope** of a variable describes where the function can be used from.

## Local Scope

A variable created inside a function belongs to the _local_ scope, and can only be used inside that function.

## Global Scope

A variable created outside a function, is called a global variable and belongs to the _global_ scope.
It can therefore be used by anyone.
