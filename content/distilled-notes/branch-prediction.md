# BRANCH PREDICTION

---

Branch prediction is a technique used by processors to guess the outcome of a conditional branch instruction before it is actually resolved.
Since the processor pipeline needs to know which instruction to execute next, waiting for the branch condition to be evaluated can cause delays ([[pipeline stalls]]).
Branch prediction helps avoid these delays by speculatively choosing the most likely path of execution.
