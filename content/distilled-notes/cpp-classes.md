# CLASSES IN C++

---

A class is a user-defined datatype that can be used in the program.
It consists of a declaration and contains both objects and [[distilled-notes/cpp-methods\|cpp-methods]].

```cpp
class Foo {
	public:
		// class attributes & methods
	private:
		// private attributes & methods
}
```

Every class is first initialised with a [[distilled-notes/cpp-class-constructor\|constructor]].
Use [[cpp-access-specifiers]\|Access Specifiers]] to define the access of the attributes and methods from outside the class itself.
