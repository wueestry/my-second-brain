# ASSEMBLER

---

The assembler is a type of systems software responsible for translating [[assembly language]] into [[binary language]] that the computer hardware can execute.
This process creates an object file, which contains machine instructions, data, and information needed for proper memory placement.
The assembler also uses conventions for handling functions, including saving register contents, managing arguments, and handling the [[stack]].
It also handles concepts like [[conditional branches]] and [[interrupts]].
