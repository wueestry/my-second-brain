# _CONST_ KEYWORD

---

_Const_ (constant) is a type qualifier, that indicates that the variable is read-only.

## Rules

- Const variables can't be left un-initialised at the time of the declaration.
- No value can be assigned anywhere in the program.
- An explicit value needs to be provided to the constant value at the time of declaration.
