# AUTOREGRESSIVE MODEL

---

An autoregressive model is a type of statistical model, which represents outputs based on previous output values. Using a linear combination of the previously obtained outputs the current output can then be predicted.

$$
y(t) = c + a_1y(t-1) + a_2y(t-2) + ...+a_py(t-p) + e(t)
$$
