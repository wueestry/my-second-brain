# MIPS INSTRUCTION SET

---

The MIPS (Microprocessor without Interlocked Pipelined Stages) instruction set is a [[RISC]] architecture, characterised by a simple and streamlined design, that avoids dependencies and stalls between instructions.
It features 32 general-purpose registers and employs a [[inbox/load-store-architecture\|load-store architecture]].

## Instruction Format

To maintain simplicity and fixed instruction length (32 bits), MIPS uses a few distinct instruction formats

### R-type (Register-type)
Used for arithmetic and logical operations, that operate on registers.

### I-type (Immediate-type)
Used for data transfer, immediate arithmetic and logical operations, and conditional branching.

### J-type (Jump-type)
Used for unconditional jumps and jump-and-link instructions.
