# PHASE PLOT

---

## Bode Phase Plot

The **phase plot** displays the phase angle ($\phi$) of the transfer function as a function of the logarithm of frequency.

$$
\phi(\omega) = \angle H(j\omega) = arctan(\frac{Im\{H(j\omega)\}}{Re\{H(j\omega)\}})
$$

Unlike the magnitude, the phase is plotted on a linear scale (typically in degrees) on the vertical axis, while the horizontal axis remains logarithmic for frequency;
